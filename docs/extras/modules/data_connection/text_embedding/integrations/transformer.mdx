# Transformer

The `TransformerEmbeddings` class uses the Transformers.js API to generate embeddings for a given text.

## Pre-requisites

### Peer Dependencies

```bash npm2yarn
npm install @xenova/transformers
```

## Example

```typescript
import { TransformerEmbeddings } from "langchain/embeddings/hf_transformers";

const embeddings = new TransformerEmbeddings({
  modelName: "Xenova/all-MiniLM-L6-v2", // In Node.js defaults to process.env.OPENAI_API_KEY
});
```
